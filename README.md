# Récupération du projet

```
git clone https://github.com/emanuelapalazzolacanfora/groupomania
cd groupomania
```

# Installation

## Frontend
```
cd frontend
npm install
```

## Backend
```
cd backend
npm install
```

## Base de données
```
npm run init
```

# Démarrage
```
node server
```

Le serveur est lancé sur le port 3008
Lancer le fichier index.html dans le navigateur
